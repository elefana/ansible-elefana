ansible-elefana
=========

Ansible role for installing Elefana.

Requirements
------------

The role assumes you have installed and configured the correct PostgreSQL and Citus prior to running.

Example Playbook
----------------

```

    - hosts: servers
      roles:
         - ansible-elefana
```
